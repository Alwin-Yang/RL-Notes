# Advanced Topics

Topics that extend the standard online, model-free RL setting.

## Model-based RL

Learn or use a dynamics model to plan, simulate data, or improve sample efficiency.

## Offline RL

Learn from a fixed dataset without further environment interaction.

## Safe RL

Add explicit safety constraints or risk objectives alongside task return.
